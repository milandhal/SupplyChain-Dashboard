# 📊 Supply Chain Analytics Dashboard (Power BI)

## 📌 Overview

This project presents a comprehensive **Supply Chain Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard is designed to provide actionable insights into supply chain performance, enabling data-driven decision-making for businesses.

It visualizes key metrics such as inventory levels, supplier performance, order trends, and logistics efficiency.

---

## 🎯 Objectives

* Monitor and optimize supply chain operations
* Identify bottlenecks and inefficiencies
* Improve inventory management
* Enhance supplier evaluation
* Support strategic decision-making with data insights

---

## 🧰 Tools & Technologies

* **Power BI Desktop (.pbix)**
* **Data Cleaning & Transformation (Power Query)**

## 📊 Key Features

### 1. 📦 Inventory Analysis

* Real-time inventory tracking
* Stock level categorization
* Identification of overstock and understock items

### 2. 🚚 Logistics & Delivery Insights

* Delivery performance metrics
* Shipment delays analysis
* Transportation efficiency tracking

### 3. 🏭 Supplier Performance

* Supplier rating and reliability analysis
* Lead time comparison
* Cost vs performance evaluation

### 4. 📈 Sales & Demand Trends

* Demand forecasting indicators
* Seasonal trend analysis
* Product-wise sales performance

### 5. 📉 KPI Dashboard

* Key Performance Indicators (KPIs)

  * Order fulfillment rate
  * Average delivery time
  * Inventory turnover ratio
  * Supply chain cost

## ⚙️ How to Use

1. Download or clone this repository:

   ```bash
   git clone https://github.com/your-username/supply-chain-analytics.git
   ```

2. Open the `.pbix` file using **Power BI Desktop**

3. Refresh the dataset if required:

   * Go to **Home → Refresh**

4. Interact with filters, slicers, and visuals to explore insights
